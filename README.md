# TestOpenGL
半年前，为了学习OpenGL，我买了一套吴亚峰编的《OpenGL ES 2.0游戏开发》，这套书分上下两卷，上卷为基础技术和典型案例，下卷为高级特效和典型案例。

这套书对OpenGL的理论讲解并不深入，大部分地方都是直接让读者看代码理解，非常不友好，目前对我来说它唯一的用处就是例子比较丰富吧。写程序的都知道，仅仅看代码而自己不动手，是根本没办法理解并掌握的。这套书的例子全是基于android的，而我目前工作主要基于Cococs2d-x，书中的代码没办法直接拿过来用。于是就有了重写例子的念头。

TestOpenGL的结构模仿Cocos2d-x的官方示例TestCpp，力求包含书中所有的例子，每个例子都会标记原书中例子的文件名以及用途，方便索引。目前看来这是一个挺大的工程，我会持之以恒的把它完成。
